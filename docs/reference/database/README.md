## Основные таблицы

- `account` — аккаунты (тенанты)
- `account_member` — участники аккаунта
- `bot` — Telegram боты
- `course` — курсы
- `course_element` — элементы курсов из БД
- `course_group` — группы (связь ботов и курсов)
- `invite_link` — пригласительные ссылки
- `schedule` — расписания для групп
- `conversation` — история взаимодействий
- `run` — сессии прохождения курсов
- `waiting_element` — отложенные элементы
- `courseparticipants` — контроль доступа
- `bannedparticipants` — блокировки пользователей

## SaaS мультитенантность

Все таблицы содержат `account_id` для изоляции данных между аккаунтами.
